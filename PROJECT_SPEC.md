# Cond Tools

Version: 1.0

---

# 1. Project Overview

Cond Tools adalah platform kumpulan online tools gratis yang membantu pengguna menyelesaikan berbagai pekerjaan langsung dari browser.

Tahap pertama berfokus pada kebutuhan UMKM Indonesia.

Ke depannya platform dapat berkembang ke tools SEO, Blogger, Designer, Developer, Office, Education, dan Productivity.
