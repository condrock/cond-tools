# Cond Tools

> Fast, free, and useful online tools for everyone.

## About

Cond Tools adalah kumpulan tools online gratis yang dibuat untuk membantu pengguna menyelesaikan pekerjaan sehari-hari dengan cepat langsung dari browser.

Website ini dibangun sebagai static website tanpa backend sehingga ringan, aman, dan dapat diakses dari berbagai perangkat.

---

## Vision

Membangun platform tools online gratis yang cepat, sederhana, dan benar-benar bermanfaat.

Target jangka panjang:

- 100% Gratis

## Ads

Website menggunakan Google AdSense.

Aturan:

- Maksimal satu slot iklan pada halaman tools.
- Iklan ditempatkan setelah hasil atau FAQ.
- Tidak mengganggu penggunaan tools.
- Tidak menyebabkan layout bergeser.
- Pengalaman pengguna lebih diutamakan daripada jumlah iklan.

## Code Style

- Kode mudah dibaca.
- Modular.
- Mudah dikembangkan.
- Tidak mengulang kode yang sama.
- Menggunakan komentar seperlunya.
- Penamaan file konsisten.

---

## Development Rules

Saat menambahkan tools baru:

- Jangan mengubah struktur website.
- Jangan mengubah Design System.
- Jangan mengubah komponen global.
- Gunakan komponen yang sudah tersedia.
- Gunakan JavaScript murni.
- Hindari library jika tidak benar-benar diperlukan.

---

## AI Development Rules

Setiap perubahan kode harus:

- Mempertahankan arsitektur yang sudah ada.
- Tidak menghapus fitur yang telah berfungsi.
- Tidak mengubah desain tanpa diminta.
- Tidak menambahkan library baru tanpa alasan yang jelas.
- Tidak menggunakan backend, API, atau database.
- Menjelaskan perubahan yang dilakukan.
- Menjelaskan file yang diubah.
- Menjelaskan alasan perubahan.
## Long-Term Goal

Menjadi salah satu website tools online gratis terbesar di Indonesia yang membantu pengguna menyelesaikan pekerjaan sehari-hari secara cepat, sederhana, dan efisien.
